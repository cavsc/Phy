About eKPI

About eKPI is a database that integrets 7 predicted kinase-substrate site interactions predicted by different models, quantitative phosphorylated proteome data from qPTM, and cancer-related clinical multiomics data collated in 13 papers in Homo sapiens. The database has its own file for each substrate and contains the above information.
